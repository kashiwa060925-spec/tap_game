# パッチノート（初期導入）
日付: 2026-01-21

## 追加
- DOMベースのタップRPGスターター（タイトル→ゲームの2シーン）
- 触る場所：`/src/rules/*`（式・定数・ステータス）、`/src/data/*`（敵・遭遇・言語）
- セーブ/ロード: localStorage（`taprpg_save_v1`）
- 外部CSS: `styles/styles.css`

## 操作
- クリック/タップのみ（攻撃/セーブ/はじめから）

## 多言語
- 日本語を既定。一部 `data/locale_*.json` を同梱（現状はJS内にフォールバック実装）

## 注意
- `index.html` を直接開く場合、ブラウザの制約で JSON の読み込みに失敗する場合があります。その際はローカルサーバ（例: `python -m http.server`）で `http://localhost:8000` を開いてください。失敗時はフォールバックのデータで動作します。
